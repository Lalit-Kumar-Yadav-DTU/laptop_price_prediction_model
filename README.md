# 💻 Laptop Price Prediction Model

A supervised machine learning project that predicts laptop prices based on various specifications such as brand, storage, weight, display features, and operating system.

---

## 📌 Overview

This project focuses on building a regression model that can estimate the price of a laptop using its key attributes. It helps in understanding how different features influence pricing in the laptop market.

---

## 🚀 Features

* Predicts laptop prices using ML regression techniques
* Handles real-world dataset with preprocessing
* Feature engineering for better model performance
* Visualization of data insights

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** Scikit-Learn, Pandas, NumPy, Matplotlib
* **Tools:** Jupyter Notebook

---

## 📊 Dataset

The dataset contains multiple laptop specifications such as:

* Brand
* RAM & Storage
* Weight
* Screen size & resolution
* Operating System
* Price (Target Variable)
